1) Utilizar como base o layout disponível no link https://www.figma.com/file/JjduV2Tg713TzYUUsees8b/efood?type=design&node-id=0-1&mode=design&t=fWWekknoLsPfkJ5n-0;

6) Subir as alterações no repositório;
